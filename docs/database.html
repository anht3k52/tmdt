ER Diagram – Hiện trạng dự án

Dựa trên mã nguồn hiện có (EF Core): bảng Students, Courses, Semesters, Enrollments và các quan hệ.

erDiagram
  STUDENTS {
    INT Id PK
    STRING StudentCode
    STRING FullName
    DATE DateOfBirth
    STRING ClassName
  }

  COURSES {
    INT Id PK
    STRING CourseCode
    STRING Name
    INT Credits
  }

  SEMESTERS {
    INT Id PK
    STRING Code
    STRING Name
  }

  ENROLLMENTS {
    INT Id PK
    INT StudentId FK
    INT CourseId FK
    INT SemesterId FK
    DOUBLE Grade
  }

  STUDENTS ||--o{ ENROLLMENTS : registers
  COURSES  ||--o{ ENROLLMENTS : has
  SEMESTERS||--o{ ENROLLMENTS : in

Ghi chú

  • Khóa duy nhất: Student.StudentCode, Course.CourseCode, Semester.Code (theo cấu hình ModelBuilder).
  • Ràng buộc duy nhất đăng ký: (StudentId, CourseId, SemesterId).
